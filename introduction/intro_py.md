## コメント化
#の後はコメント扱い．実行時に無視される．
```python
# a = 1
```
複数行のコメントはダブルクォーテーション，またはシングルクォーテーションの3連で囲む．
```python
"""
a = 1
こんにちは
"""
```
```python
'''
a = 1
こんにちは
'''
```
主に関数やクラスのドキュメンテーション文字列に使う．  
vscodeだと  
コメント化したい部分を選択 ->  
Ctrl + k ->  
Ctrl + c  
で一気にコメント化できる．
