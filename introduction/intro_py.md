## コメント化
#の後はコメント扱い．実行時に無視される．
```python
# a = 1
```
vscodeだと  
* コメント化したい部分を選択 ->
* Ctrl + k ->
* Ctrl + c
で使用言語に依らず一気にコメント化できる（便利）．  

複数行のコメントはダブルクォーテーション，またはシングルクォーテーションの3連で囲む．
```python
"""
a = 1
こんにちは
"""
```
```python
'''
a = 1
こんにちは
'''
```
主に関数やクラスのドキュメンテーション文字列に使う．
